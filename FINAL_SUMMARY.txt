📞 WHEN THINGS GO WRONG:

  "Quote form doesn't work"
  → Don't modify QuoteForm.tsx or api/quote/route.ts
  → Check form field names match
  → Check browser console for errors

  "Logo doesn't show"
  → Files must be named exactly logo.png and icon.png
  → Must be in public/ folder
  → Delete .next/ and rebuild

  "Build fails"
  → You edited a component file (don't do that)
  → Delete node_modules, reinstall
  → Run npm run build to see exact error

  "Deploy fails on Vercel"
  → Make sure all imports resolve
  → Check tsconfig.json is valid
  → Make sure package.json has Next.js version
